# Cricket Live Dashboard and Analytics

A simple React web application that shows live cricket scores and match-related information in a clean dashboard layout.

Live Website:
https://sanjay26445.github.io/cricket-live-dashboard-and-analytics/

## Technologies Used
- React (Vite)
- JavaScript
- CSS
- GitHub Pages

## Features
- Live cricket score display
- Simple and responsive UI
- Fast performance using Vite
- Deployed using GitHub Pages

## Run Project Locally

1. Clone the repository
git clone https://github.com/sanjay26445/cricket-live-dashboard-and-analytics.git

2. Go to project folder
cd cricket-live-dashboard-and-analytics

3. Install dependencies
npm install

4. Run the app
npm run dev

## Build Project
npm run build

## Deploy
npm run deploy
